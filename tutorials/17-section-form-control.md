# 17 Section Form Control

Goal: move a patch through sections like verse and chorus.

Patch: [examples/section_controlled.pulse](/Users/md/Downloads/plugin%20language/examples/section_controlled.pulse)

Concepts:

- `generate section` creates named formal blocks.
- another clock or control stream can advance the form.
- a projector can react differently depending on the current section.

Key idea:

Fabric is not limited to short loops. It can carry a higher-level song form.
